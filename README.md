# NerdDevs-Frontend

## Live-Site: (https://nerddevs-frontend.vercel.app/)

## Features

- User registration
- User account activation
- User login
- Protected routing

## Run Locally

Clone the project:

```bash
https://github.com/hasanshanto01/nerddevs-frontend.git
```

Go to the project directory

```bash
cd nerddevs-frontend
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
